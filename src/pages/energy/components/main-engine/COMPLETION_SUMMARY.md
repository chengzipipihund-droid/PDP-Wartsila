## ✨ 核心功能模块

### 1. **EngineDetailPanel** (详细面板)
显示当前活跃引擎的完整信息

```
┌─────────────────────────────┐
│ ME1  Wärtsilä 46F   [RUN]   │  ← 标题栏
├─────────────────────────────┤
│ [Icon] [Arrows]       [Bar] │  ← 线稿 + 能量箭头 + 容量
│                        75%  │
│                         [✕] │  ← 关闭按钮
└─────────────────────────────┘
```

**显示内容:**
- ✅ 引擎编号 (ME1 / ME2 / ME3 / ME4)
- ✅ 引擎型号 (Wärtsilä 46F)
- ✅ 运行状态 (RUN / STOP / FAULT) - 彩色标签
- ✅ 引擎线稿图标 - 简洁的 SVG 线条画
- ✅ 能量流向指示器:
  - 🟨 **黄色箭头** - 正常能量流失 (下向)
  - 🔴 **红色箭头** - 快速能量流失 (下向，闪烁)
  - 🟢 **绿色箭头** - 能量输入 (上向)
- ✅ 右侧容量条 - 垂直进度条 (0-100%)
  - 渐变绿色填充
  - 闪烁动画效果
  - 百分比显示

### 2. **EngineCompactPanel** (折叠面板)
显示其他三个引擎的简洁视图

```
┌────────────┐
│ ME2 [RUN]  │  ← 编号 + 状态
│     ●      │  ← 能量指示灯
└────────────┘
```

**显示内容:**
- ✅ 引擎编号
- ✅ 运行状态 - 彩色标签
- ✅ 能量指示灯 - 单个点号显示能量状态
  - 🔴 红色 = 快速流失 (urgent)
  - 🟨 黄色 = 正常流失 (normal)
  - ⚫ 灰色 = 空闲 (idle)
- ✅ **可点击切换** - 点击时与详细面板互换

### 3. **MainEnginePanel** (主容器)
管理所有引擎的状态和布局

```
┌────────────────────────────────────┐
│ 左侧: 详细面板          右侧: 折叠面板│
│ ┌──────────────┐    ┌──────────┐  │
│ │   ME1详细    │    │ ME2 [RUN]│  │
│ │              │    ├──────────┤  │
│ │   75%        │    │ ME3 [STOP│  │
│ │              │    ├──────────┤  │
│ │         [✕]  │    │ ME4 [RUN]│  │
│ └──────────────┘    └──────────┘  │
└────────────────────────────────────┘
```

## 🎯 交互流程

### 场景1: 初始加载
1. ME1 自动显示在详细面板
2. ME2、ME3、ME4 显示为折叠面板（右侧竖排）
3. 用户可以点击任意折叠面板

### 场景2: 切换引擎
```
用户点击 ME3 (折叠面板)
  ↓
ME3 切换到详细面板（左侧）
  ↓
原来的 ME1 显示为折叠面板（右侧）
  ↓
平滑动画 (CSS transition)
```

### 场景3: 关闭详细面板
```
用户点击 关闭按钮 [✕]
  ↓
第一个可用的折叠面板显示为详细面板
  ↓
原来的详细面板消失
```

## 🎨 样式亮点

### 色彩系统
- **运行中** (RUN): 🟢 `#4FBF65` 绿色
- **停止** (STOP): ⚫ `#CCCCCC` 灰色
- **故障** (FAULT): 🔴 `#E74C3C` 红色（带脉冲动画）
- **正常流失**: 🟨 `#EAB71D` 黄色
- **快速流失**: 🔴 `#E74C3C` 红色（带脉冲）
- **能量输入**: 🟢 `#4FBF65` 绿色

### 动画效果
- ✨ 容量条填充: 弹性 cubic-bezier 动画
- ✨ 容量条闪烁: shimmer 效果
- ✨ 红色箭头: pulse 脉冲增大动画
- ✨ 故障状态: 闪烁透明度动画
- ✨ 悬停效果: 图标放大、按钮亮度增加

### 响应式设计
- **桌面 (>1024px)**: 详细面板占大部分，折叠面板竖排120px宽
- **平板 (768-1024px)**: 折叠面板宽度减少到100px
- **手机 (<768px)**: 详细面板上，折叠面板横排排列
